# Triage Labels

SDD skills 用五个 canonical triage 角色交流。本文件记录本仓库 issue 文件实际使用的字符串。

| Role              | 本仓库 label       | 含义                           |
| ----------------- | ------------------ | ------------------------------ |
| `needs-triage`    | `needs-triage`     | 需要维护者评估                 |
| `needs-info`      | `needs-info`       | 等待报告人补充信息             |
| `ready-for-agent` | `ready-for-agent`  | 已完全规格化，可交给 AFK agent |
| `ready-for-human` | `ready-for-human`  | 需要人工实现                   |
| `wontfix`         | `wontfix`          | 不予处理                       |

当某个 skill 提到某个角色（如 "apply the AFK-ready triage label"），用右列对应的字符串。本仓库现有 spec 已在使用 `ready-for-agent` 等。

需要改用别的词汇时，编辑右列。
